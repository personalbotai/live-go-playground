# Live Go Playground

Interactive Go code editor and runtime with real-time execution.

## Features

- Write and execute Go code in browser
- Real-time output
- Share code snippets
- Syntax highlighting

## Development

This is a static single-page application. Simply open `index.html` in your browser or deploy to GitHub Pages.

## Deployment

GitHub Pages is configured via `.github/workflows/deploy.yml`. After enabling Pages in repository settings, the site will be available at:

```
https://personalbotai.github.io/live-go-playground/
```
